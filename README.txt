Dropler GUI by araan515
===================

Установка:
1. Распакуйте архив в любую папку
2. Заполните файл task_1/Configs/log_pass.txt строками в формате login1:password1 (каждый новый аккаунт с новой строки)
3. Поместите .maFile файлы в папку task_1/maFiles
4. Переименуйте DroplerGUI.bin в DroplerGUI.exe
5. Запустите DroplerGUI.exe
6. При создании нового потока шаги №2 и №3 повторить для соотвсетвующей папки Task_N

Требования:
- Windows
- .NET 8.0 или выше (https://dotnet.microsoft.com/download/dotnet/8.0)
- Steam аккаунт с настроенным мобильным аутентификатором


Поддержка:
- Исходный код: https://github.com/Pirozhok40/Dropler-GUI/
- Обсуждения и проблемы: https://github.com/Pirozhok40/Dropler-GUI/issues
